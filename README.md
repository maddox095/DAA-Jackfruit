# DAA-Jackfruit

This repository contains three Dynamic Programming and Greedy algorithm write-ups with matching C implementations.

## Problem 1: Activity Selection

See [Problem1.md](Problem1.md) and [problem1.c](problem1.c).

This problem uses a greedy strategy to select the maximum number of mutually compatible activities by always choosing the activity that finishes earliest. The implementation prints the selected activities and count for any testcase you paste into `main`.

## Problem 2: Closest Pair of Points

See [Problem2.md](Problem2.md) and [problem2.c](problem2.c).

This problem uses divide and conquer to find the minimum distance between two points in the plane. The program prints the input points and the closest distance for any testcase you paste into `main`.

## Problem 3: Edit Distance

See [Problem3.md](Problem3.md) and [problem3.c](problem3.c).

This problem uses dynamic programming to compute the minimum number of insert, delete, and replace operations needed to convert one string into another. The program prints the minimum edit distance, the DP table, and one valid sequence of operations.

## How to Run

- Problem 1: compile and run [problem1.c](problem1.c)
- Problem 2: compile and run [problem2.c](problem2.c)
- Problem 3: compile and run [problem3.c](problem3.c)
